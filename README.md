# TimeOutChef Recipe Site (GitHub Pages)

This folder is ready to publish as a GitHub Pages site using Jekyll.

## Publish (quick)
1. Create a new GitHub repo (e.g. `timeoutchef-recipes`)
2. Upload the contents of this folder to the repo (root level).
3. In GitHub: Settings → Pages
   - Source: Deploy from a branch
   - Branch: `main` / `/ (root)`
4. Open the Pages URL GitHub provides.

## Custom domain
In GitHub: Settings → Pages → Custom domain, then follow the DNS instructions.
